Website Module

School Events

Stay connected with the school community. This module keeps parents and students updated on all academic and extracurricular activities.

Upcoming Events: Automated list of future events with venue details and countdowns.
Calendar Embed: Full Google Calendar integration for seamless personal scheduling.
Past Events Carousel: Interactive sliders showcasing memories from completed events.

How it Works

Smart Sorting: Once an event date passes, the system **automatically moves** it from 'Upcoming' to the 'Archive' section, keeping your page clean and relevant.
